# HVAC Designer SVG icon palette

This folder contains the first CoreUI icon palette in SVG format.

Files:

- `hvac-icons-dark.svg`: muted duotone palette for the current dark CoreUI.
- `hvac-icons-light.svg`: same geometry with stronger colors for the planned light CoreUI.

Both SVG files contain reusable symbols with stable ids:

- Header actions: `new-project`, `open-project`, `save-project`, `project-properties`, `settings`, `export`, `print-report`, `info`, `import`, `search`
- General/navigation: `dashboard`
- Building thermal: `building-energy`, `building`, `u-value`, `heat-load`, `certification`
- Heating/cooling hydraulics: `heating`, `cooling`, `pipe-network`, `pump`
- Sanitary: `sanitary-water`, `water-demand`, `sewer`, `dhw-circulation`, `long-profile`
- Air systems: `air`, `air-velocity`, `duct-sizing`, `duct-network`
- Flue gas/safety: `flue-gas`, `safety`

Design direction:

- Dark theme: low-opacity colored fills with clear colored strokes.
- Light theme: same icon shapes, slightly stronger and more colorful fills/strokes.
- All icons use a 24x24 viewBox and are designed to render cleanly at 20-32 px.
